**Здесь 3 блокнота, 2 по CV, 1 по NLP**

Итог работы **CV** 71-73% accurancy, классификация 38 классов фигурок лего. Сделал векторизацию картинок с помощью resnet50. Причиной относительно невысокого процента - это маленький объем датасета. Однозначно можно улучшить, вероятно позже вернусь с этой целью.

Итог работы **NLP** 97% accurancy, классификация тональности комментариев (всего 4 класса). Применил LogisticRegression и Catboost, lg быстро и сердито, cb просто сердито, но оч долго относительно lg, в данном случае он не нужен такой. Применил лемматизацию и метод TF-IDF и минимальный EDA. Lg 97% & cb 95% на валидации.

Для работы использовал marimo для блокнотов, scikit-learn, pytorch, pandas, numpy, nltk, catboost, torchvision, pillow
